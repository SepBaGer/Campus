# Workspace

Este directorio guarda el estado local de trabajo: planes, bitacoras, reportes y
artefactos generados durante tareas activas.

## Reglas

- No es codigo de producto.
- No es una fuente de verdad para despliegue.
- Cada tarea debe mantener aqui su `plan.md`, `tasklog.md`, `changelog.md` y
  `artifacts/` cuando la automatizacion de JM-ADK este habilitada.
