# Tools

`jm-adk` que gobierna la operacion del repositorio.

## Regla principal

- Trata `tools/jm-adk/` como fuente de verdad metodologica.
- No edites `tools/jm-adk/` salvo que la tarea sea cambiar el framework mismo.
