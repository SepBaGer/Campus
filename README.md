# Campus

Aplicacion web del nuevo Campus MetodologIA, preparada para despliegue como frontend estatico con Supabase remoto.

## Contenido del repositorio

- `campus-v2/`: aplicacion Vite del Campus
- `campus-v2/supabase/`: migraciones, funciones y semilla base para Supabase

## Despliegue

La guia operativa principal esta en [campus-v2/README-DEPLOY.md](./campus-v2/README-DEPLOY.md).

## Variables de entorno

Usa estos archivos como base:

- `campus-v2/.env.example`
- `campus-v2/.env.production.example`
