# campus-platform

Superficie activa del Campus en Astro + Lit.

## Objetivo

Servir como capa principal del producto para:

- sitio publico
- detalle de programa
- acceso y activacion
- portal del estudiante
- operacion interna
- verificacion publica de credenciales

## Scripts

- `npm run dev`
- `npm run test`
- `npm run test:a11y`
- `npm run test:perf`
- `npm run build`
- `npm run preview`

## Variables de entorno

Usa [`.env.example`](./.env.example) como punto de partida.
Para build remoto usa tambien [`.env.production.example`](./.env.production.example).
`PUBLIC_CAMPUS_PLATFORM_SSO_CONNECTIONS` acepta un JSON array opcional para
publicar botones curados de SSO enterprise en `/acceso`; si no existe, la UI
igual permite discovery por correo corporativo, dominio o `providerId`.
